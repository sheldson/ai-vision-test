测试卡 L2-M-04 跨会话文档记忆

级别/维度: L2 / 记忆维度

测试目的: 评估玩偶在多个会话间记忆文档信息的能力

所需材料:

  • 包含日期相关信息的文档（周计划、课程表等）

执行步骤:

  1. 在第一次会话中展示文档
  2. 确保玩偶理解文档内容
  3. 结束会话，至少间隔2小时以上
  4. 开始新会话，询问之前文档中的信息 （"今天按照课程表应该上什么课？"）

评分标准:

  • 1分：完全无法跨会话记忆文档信息
  • 2分：极少能回忆基本信息片段
  • 3分：能回忆部分关键信息但有错误
  • 4分：能准确回忆大部分关键信息
  • 5分：完整精确回忆文档信息并能适时主动提及相关内容

观察要点:

  • 长时记忆的保持能力
  • 不提示情况下回忆的准确性
  • 主动应用记忆信息的能力
